Dijkstra (OSPF) algorithm demo

Shortest path from a vertex (node) to all other nodes in a weighted graph

- Detailed description of the algorithm: Wikipedia
- Choose the number of nodes, a random graph (network with random links & costs) will be generated.
- Click Draw to draw (or re-draw) a graph, then Start to run the algorithm.
- The result is a tree (should be no loops), consisting of all shortest paths from every node to ROOT.
- Click on any node to see the shortest path to root.
- Thanks to Cytoscape.js all nodes are clickable and moveable.
